3. 个人信息的保护

我们采取适当的安全措施来保护您的个人信息不被未经授权的访问、使用或披露。

4. 信息共享

我们不会出售、出租或共享您的个人信息给第三方，除非在以下情况下：

  • 得到您的同意；
  • 遵守法律要求；
  • 保护我们的权利和财产。

5. 更新和访问个人信息

如果您对我们的隐私政策有任何疑问或需要您可以随时更新或删除您的个人信息，请通过Contact us与我们联系。
